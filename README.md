# Personal Portfolio Website

A responsive personal portfolio website built to showcase my skills, projects, education, and experience as a Software Developer.

## Features
- Responsive design for all devices
- Modern and clean user interface
- About Me section
- Skills showcase
- Projects portfolio
- Contact section
- Smooth navigation and animations

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
This portfolio serves as a central place to highlight my technical skills, academic projects, and professional journey, making it easier for recruiters and collaborators to learn more about me.
